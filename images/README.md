# Images Folder

Place your Funko Pop and Action Figure images here.

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`

## Recommendations
- Use square images (1:1 aspect ratio) for best results
- Recommended size: 500x500px or larger
- Use PNG with transparent backgrounds for cleaner look
- Keep file sizes under 500KB for faster loading

## Example Filenames
- `spiderman.png`
- `ironman.jpg`
- `batman.png`
- `grogu.webp`

## How to Add Images

1. Add your image files to this folder
2. Update `script.js` with the filename:

```javascript
{
    name: "Spider-Man",
    type: "funko",
    image: "spiderman.png"  // This filename
}
```

3. Commit and push to GitHub

That's it! The website will automatically display your images.
